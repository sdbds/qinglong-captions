# 图片预处理与对齐

图片预处理工具负责缩放、裁剪、透明边处理，以及可选的参考图对齐。它适合在打标、评分或训练数据导入前统一图像尺寸和构图。

## 入口

```powershell
.\2.2.preprocess_images.ps1
```

Python 入口：`python -m utils.preprocess_datasets --help`。依赖 profile 为 `image-align`。

## 常用能力

- 限制长边、短边或总像素数。
- 按目录递归处理并保持相对结构。
- 裁切透明区域或填充指定背景色。
- 使用配对参考图进行特征对齐。
- `matcher-backend=auto` 根据运行环境选择匹配后端，并在失败时回退。

预处理会生成新文件。不要把输出写回正在读取的源目录；全量运行前用少量不同宽高比图片检查裁切与对齐结果。
